# tests-with-selenium
 
This an project to explore tests with selenium.
The repository is separated in two parts.

- Backend
- Frontend.

# Built With
- Java 8
- Spring-Boot
- seleniumhq
- H2
- Angular 8

# Prerequisites
 - Ng
 - JVM 8
 
# Usage
### Running Backend
`mvn spring-boot:run`

### Running frontEnd
`ng serve --open`

### Running tests with selenium
`mvn spring-boot:run`
